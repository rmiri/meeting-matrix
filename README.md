# Meeting Matrix

## Ruby version

``` ruby '2.7.0' ```

## Configuration

``` bundle install ```

## Database 

 I used postgres as I wanted to host the app on Heroku

 to set up make that postgres is runing on your machine

To create the data base:
 ``` rails db:create ```

To create the tables:
``` rails db:migrate ```

To add content to the tables:
``` rails db:seed ```
